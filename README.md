# youtalk-blog
Built a responsive blog from the design mockup.

As a frontend developer at YouTalk, I was responsible for turning the design mockup into a fully functional blog interface.

The main goal was to create a clean, fast, responsive, and user-friendly experience.

I implemented the blog layout according to the provided design, keeping visual consistency across all sections.

Special attention was given to responsive design, semantic HTML, reusable CSS components, and accessibility.

The result is a modern blog page that looks polished, works smoothly on different devices, and is easy to maintain.
